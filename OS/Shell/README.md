Navigation in Bash CLI
-

- **`↓ | ↑`** => iterating through command history
-  **`CTRL+R`** => searching in command history by typing charachter by charachter
-  **`TAB`** => autocompleting a command
-  **`CTRL+C`** => killing the current running process
